# Awesome AIGC

AIGC，全称是 Artificial Intelligence Generated Content，是指由 AI 生成的内容（包括文字、图片、音视频等）。

随着 AI 技术越来越强大，AIGC 的成本也越来越低，大量技术、模型、代码都已公开，也有了很多消费级应用。

这个仓库搜集了 AIGC 相关的资源，希望能帮助大家~~搞颜色~~提高工作效率，进入 AI 时代。

## 开源项目
